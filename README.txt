ESPAÑOL

Este es una aplicación para gestión de stock de librería ejecutado en terminal.
Es mi primer proyecto que realice para un curso de Programación inicial.

La aplicación consta de dos archivos, el Library_Fuctions, el cual tiene todas las 
funciones que permiten la gestión de la aplicación. Función de limpieza de pantalla, funciones de chequeo de opciones 
o consultas al usuario y todo lo relacionado con la administración del stock de una librería y consulta de reportes.

Luego está el Main_Library el cual contiene la función principal que muestra el menú principal y los distintos submenús.
También contiene un diccionario con una base de libros inicial.

La aplicación no cuenta con una base de dato, ya que al momento de realizarlo no contaba con el conocimiento para hacerlo.
La idea es a medida que me capacito ir mejorándolo.

ENGLISH
This is an inventory management application for a bookstore, executed in the terminal. 
It is my first project that I developed for an introductory programming course.

The application consists of two files: Library_Functions, which contains all the functions that enable the management of 
the application. This includes a screen clearing function, functions for checking options or user queries, and everything 
related to the administration of bookstore stock and report queries.

Then there's Main_Library, which contains the main function that displays the main menu and various submenus. 
It also includes a dictionary with an initial book database.

The application does not have a database because, at the time of its development, I did not have the knowledge 
to implement one. The idea is to gradually improve it as I gain more knowledge and skills.
